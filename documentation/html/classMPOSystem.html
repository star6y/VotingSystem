◆ allocateSeats()

void MPOSystem::allocateSeats ( )
overridevirtual

Allocates seats based on party total votes, and assigns winning candidates based on their popularity within the party.

Inherits from the abstract Class ElectionSystem, and overrides the method. This function will find the quota to do the first seat allocation. It will then move on to the second seat allocation, where it will see if there are possible ties between the parties. Handling ties will be handled by another class, this function can get the result. After all seats are allocated to the parties, we then have to find the winning candidates within the parties, and handle any ties between candidates of any specific party. To make it easier to provide results, we will move Candidate objects from the "candidates" vector into a vector of winners, "winners".

Implements ElectionSystem.
