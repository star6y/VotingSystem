Allocates seats based on party total votes, and assigns winning candidates based on their popularity within the party. 

Inherits from the abstract Class \mbox{\hyperlink{classElectionSystem}{Election\+System}}, and overrides the method. This function will find the quota to do the first seat allocation. It will then move on to the second seat allocation, where it will see if there are possible ties between the parties. Handling ties will be handled by another class, this function can get the result. Since the order of candidates in each party is predetermined, there is nothing more to do. 

Implements \mbox{\hyperlink{classElectionSystem_a1e5cb35444beb775f688d6a4299c3629}{Election\+System}}.

Builds party list with a list of their candidates. 

Inherits from the abstract Class \mbox{\hyperlink{classElectionSystem}{Election\+System}}, and overrides the method. This method will read each line, take the \mbox{\hyperlink{classParty}{Party}}, add it to \char`\"{}parties\char`\"{}, then read each \mbox{\hyperlink{classCandidate}{Candidate}} on that line. Later this will help us allocate the seats within the parties. At the same time while we read each \mbox{\hyperlink{classParty}{Party}}, we also add these parties to the \char`\"{}party\+Votes\char`\"{}, which will store their votes in a piar.
